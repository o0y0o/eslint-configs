`@0y0/babel-preset-react` is a common Babel preset for React.

## Installation

```sh
npm install @0y0/babel-preset-react --save-dev
```

## Usage

```js
{
  "presets": ["@0y0/babel-preset-react"] // or `@0y0/react`
}
```
